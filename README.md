# Stable
使用WordPress有四年了，想做一個穩定的WordPress主題，以供瀏覽方面能夠更人性化。  
此主題以兩欄式簡潔為主。

### Download Theme
<a href="https://github.com/unromanticman/stable/releases">下載Stable主題</a>

### Usage
<pre>
1.將下載完的ZIP解壓縮後放置於 /wp-content/themes/ 目錄下
2.進入WordPress啟用主題
</pre>
